⚡ נבנה עם HTML מודרני

האתר הזה משתמש בפיצ'רים נטיביים של הדפדפן — בלי frameworks, בלי JS מיותר:

  • View Transitions API — מעברים חלקים בין הפיד לעמודי המאמר
  • Popover API — הפאנל הזה והרשמה נטיביים, ללא ספריות מודל
  • CSS :has() — כרטיסים מגיבים זה לזה בהעברת עכבר
  • Container Queries — פריסה שמגיבה לרוחב הקונטיינר שלה
  • animation-timeline: scroll(root) — פס ההתקדמות למעלה
  • text-wrap: balance / pretty — עימוד טקסט חכם
